# testdata
A collection of publicly publishable test imaging data.

Images w/ in this repository have been arranged in folders by imaging format (dicom, ecat, nifti, etc) and where applicable in subfolders by manufacturer.

e.g. `dicom/siemens` or `dicom/ge` or `ecat`

Most likely mages are identified below by their location within their repo following the original source they were collected from.

[1] ecat/ECAT7_multiframe.v
[2] dicom/siemens/, dicom/GE, dicom/Phillips    https://github.com/neurolabusc/dcm_qa_fmap
